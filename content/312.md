## Code

```bash
echo '[{"foo": 1, "bar": 123}, {"foo": 1, "bar": 123}, {"foo": 4, "bar": 123}]' | jq 'unique_by(.foo)'
```

## Example

```
$ JSONSTRING='[{"foo": 1, "bar": 123}, {"foo": 1, "bar": 123}, {"foo": 4, "bar": 123}]'

$ echo "$JSONSTRING" | jq 'unique_by(.foo)'
[
  {
    "foo": 1,
    "bar": 123
  },
  {
    "foo": 4,
    "bar": 123
  }
]

$ echo "$JSONSTRING" | jq 'unique_by(.bar)'
[
  {
    "foo": 1,
    "bar": 123
  }
]
```

## Output

```
[
  {
    "foo": 1,
    "bar": 123
  },
  {
    "foo": 4,
    "bar": 123
  }
]
```
