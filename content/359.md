## Code

```bash
curl -H "Authorization: token $GITHUB_TOKEN" \
  -H "Content-Type: $(file -b --mime-type $FILE)" \
  --data-binary @$FILE \
  "https://uploads.github.com/repos/$GITHUB_OWNER/$GITHUB_REPO/releases/$GITHUB_RELEASE/assets?name=$(basename $FILE)"
```
