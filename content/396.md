## Code

```bash
cp /etc/sudoers . 

# json
cvtsudoers -f json -o sudoers.json sudoers

# combines all /etc/sudoers.d/* and /etc/sudoers
cvtsudoers -f sudoers -o sudoers.txt sudoers
```
