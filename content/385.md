## Code

```bash
$ cat letters.txt
b
B
A
c
a
C
D
d

$ LC_ALL=en_US.UTF-8 sort letters.txt
a
A
b
B
c
C
d
D

$ LC_ALL="C" sort letters.txt
A
B
C
D
a
b
c
d
```
