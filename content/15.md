## Code

```bash
ls -R | grep ":$" | sed -e 's/:$//' -e 's/[^-][^\/]*\//--/g' -e 's/^/   /' -e 's/-/|/'
```

## Example

```
$ ls -R | grep ":$" | sed -e 's/:$//' -e 's/[^-][^\/]*\//--/g' -e 's/^/   /' -e 's/-/|/'
   .
   |-inventory
   |-roles
   |---dummy
   |-----defaults
   |-----files
   |-----handlers
   |-----meta
   |-----tasks
   |-----templates
   |-----tests
   |-----vars
```
