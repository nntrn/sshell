## Code

```bash
sudo --list | grep -ws User -A 1000 | sed 's/,/\n/g' | sed 's/:/:\n/g' | grep -E '^\s+?/' | sed -e 's/^[[:space:]]*//'
```

## Example

```
$ sudo --list | grep -ws User -A 1000 | sed 's/,/\n/g' | sed 's/:/:\n/g' | grep -E '^\s+?/' | sed -e 's/^[[:space:]]*//'
/bin/docker
/bin/su
/usr/local/bin/gitlab-runner
```
