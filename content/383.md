## Code

```bash
# include tags
sed -n '/<tag/,/<\/tag/ p'

# get inner lines only
sed -n '/<ul/,/<\/ul/ {//! p}'
```

## Example

```
## include tag
$ sed -n '/<ul/,/<\/ul/ p' <<EOF
> <p>ignored text</p>
> <ul>
>   <li>one</li>
>   <li>two</li>
> </ul>
> <span>also ignored</span>
> EOF

<ul>
  <li>one</li>
  <li>two</li>
</ul>

## do not include tag
$ sed -n '/<ul/,/<\/ul/ {//! p}' <<EOF
> <p>ignored text</p>
> <ul>
>   <li>one</li>
>   <li>two</li>
> </ul>
> <span>also ignored</span>
> EOF

  <li>one</li>
  <li>two</li>
```
