## Code

```bash
CURL_GRAPHQL_QUERY="$(
  jq -c -n --arg query '{...}' '{"query":$query}'
)"
```

## Example

```
CURL_GRAPHQL_QUERY="$(
  jq -c -n --arg query '{
  projects(search: "token") {
    nodes {
      name
      description
      lastActivityAt
      webUrl
      sshUrlToRepo
      fullPath
      pipelineCounts {
        all
        finished
        pending
        running
      }
      languages {
        name
      }
    }
  }
}' '{"query":$query}'
)"

curl https://gitlab.com/api/graphql -X POST -H "Content-Type: application/json" -H "Authorization: Bearer $GITLAB_TOKEN" --data "$CURL_GRAPHQL_QUERY"
```
