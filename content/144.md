## Code

```bash
awk -F\' /^menuentry/{print\$2} /etc/grub2.cfg | grep -vE "(uek|rescue)" | sort | tail -n 1
```

## Example

```
$ awk -F\' /^menuentry/{print\$2} /etc/grub2.cfg | grep -vE "(uek|rescue)" | sort | tail -n 1
Oracle Linux Server 7.9, with Linux 3.10.0-1160.36.2.el7.x86_64
```
