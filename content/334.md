## Code

```bash
# get values
echo '{ "a": 1, "b": 2 }' | jq '.[]'

# get keys
echo '{ "a": 1, "b": 2 }' | jq 'keys | .[]'
```

## Example

```
$ echo '{ "a": 1, "b": 2 }' | jq '.[]'
1
2

$ echo '{ "a": 1, "b": 2 }' | jq 'keys | .[]'
"a"
"b"
```
