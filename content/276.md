## Code

```powershell
$Self = Get-ADUser -Identity $Env:Username -Properties *
$EID = $Self.EmployeeId

Get-ADUser  -Filter "OwnerID -like $EID"  -Properties * | sort-object -property PasswordLastSet | Format-Table Name,PasswordLastSet -AutoSize -Wrap
```
