## Code

```bash
jq -Rn '[(inputs | split(" ")) | .[] | split("=") | {key: .[0], value: .[1]}] | from_entries'
```

## Example

```
$ echo 'a=3 4=9' | jq -Rn '[(inputs | split(" ")) | .[] | split("=") | {key: .[0], value: .[1]}] | from_entries'
{
  "a": "3",
  "4": "9"
}
```
