## Description

use the logical and && and or || to compare strings:

## Code

```bash
[[ "string1" == "string2" ]] && echo "Equal" || echo "Not equal"
```

## Example

```
$ [[ "string1" == "string1" ]] && echo "Equal" || echo "Not equal"
Equal

$ [[ "string1" == "string2" ]] && echo "Equal" || echo "Not equal"
Not equal
```
