## Code

```bash
echo "A,B,C,D,E,F,G,H,I,J,K,L,M,N,O" | awk -v RS='[,\n]' '{a=$0;getline b; getline c; print a,b,c}' OFS=,
```

## Output

```
A,B,C
D,E,F
G,H,I
J,K,L
M,N,O
```
