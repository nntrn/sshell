## Code

```javascript
css = new Set(
  [
    ...$$('style[data-load-themed-styles]')
      .map(e =>
        e.innerText
          .replace(/(\u007D)/g, '$1\n')
          .split('\n')
          .filter(Boolean)
          .filter(p => p.indexOf('media') < 0)
          .filter(p => !/^[^_]+_[\d\w]+\s?{/.test(p))
          .filter(p => !/.*[\[\]].*/.test(p))
      )
      .flat(2),
  ]
    .filter(e => e.trim())
    .map(e => e.replace(/\n/g, '').split('{'))
    .map(c => c[0].split(',').map(a => a + '\u007B' + c[1]))
    .flat(2)
    .sort()
)
```
