## Code

```bash
docker inspect --format '{{range .NetworkSettings.Networks}} {{.IPAddress}} {{end}}' $CONTAINER_NAME
```

## Example

```
$ docker inspect --format '{{range .NetworkSettings.Networks}} {{.IPAddress}} {{end}}' awx_web
172.24.0.4
```
