## Code

```bash
openssl req -new -newkey rsa:4096 -nodes -keyout "private.key" -out "$(hostname -f).csr" -subj "/C=US/ST=Texas/L=Round Rock/O=example/OU=Digital/CN=$(hostname -f)"
```
