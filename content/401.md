## Code

```bash
jq 'group_by(.language)| map( {"\(.[0].language)":length}) |add'
```

## Example

```
$ curl -s https://api.github.com/users/nntrn/repos | jq 'group_by(.language)| map( {"\(.[0].language)":length}) |add'
{
  "null": 7,
  "HTML": 4,
  "Java": 1,
  "JavaScript": 11,
  "Jinja": 2,
  "R": 1,
  "SCSS": 1
}
```
