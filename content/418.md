## Code

```bash
# identify required/optional keys are used in all objects in array
jq 'map(to_entries|map(.key)[])|group_by(.)|map({"\(.[0])": length })|add'

# count tag items
jq 'map(.tags[])|group_by(.)|map({"\(.[0])": length })|add'
```
