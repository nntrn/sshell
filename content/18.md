## Code

```bash
find . | grep -E "(__pycache__|\.pyc|\.pyo$)" | xargs rm -rf
```

## Example

```
alias pyclear='find . | grep -E "(__pycache__|\.pyc|\.pyo$)" | xargs rm -rf'
```
