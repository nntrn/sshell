## Code

```bash
getsame() {
  string1=${1:?}
  string2=${2:?}
  first_diff_char=$(cmp <(echo "$string1") <(echo "$string2") | cut -d " " -f 5 | tr -d ",")
  echo ${string1:0:$((first_diff_char - 1))}
}
```

## Example

```
$ path1=kv/SERVICE_ACCOUNT/SQLSERVER
$ path2=kv/SERVICE_ACCOUNT/MSSQL
$ getsame $path1 $path2
kv/SERVICE_ACCOUNT/
```
