## Description

fast solution that uses filter-branch instead of rebase

clean up with:
git reflog expire --expire=now
git gc --prune=now

## Code

```bash
git filter-branch --commit-filter '
    if [ `git rev-list --all --grep "<PATTERN TO REMOVE>" | grep -c "$GIT_COMMIT"` -gt 0 ]
    then
        skip_commit "$@";
    else
        git commit-tree "$@";
    fi' HEAD
```
