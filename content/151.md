## Code

```bash
realpath --relative-to=$DIR1 $DIR2
```

## Example

```
$ realpath --relative-to=/home/annie_tran /tmp
../../tmp

$ realpath --relative-to=/tmp /home/annie_tran
../home/annie_tran
```
