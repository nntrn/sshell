## Code

```bash
sed ':a;N;$!{/\n$/!ba}; s/[[:blank:]]*\n[[:blank:]]*/ /g' textfile.txt
```

## Example

```
# unwrap sudo.log file
sed ':a;N;$!{/\n$/!ba}; s/[[:blank:]]*\n[[:blank:]]*/ /g' /var/log/sudo.log | sed 's/COMMAND/\nCOMMAND/g'
```
