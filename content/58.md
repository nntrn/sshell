## Code

```bash
alias c1="awk '{print \$1}'"
alias c2="awk '{print \$2}'"
alias c3="awk '{print \$3}'"
alias c4="awk '{print \$4}'"
alias c5="awk '{print \$5}'"

```

## Example

```
demo=' a1 a2 a3
b1 b2 b3
c1 c2 c3'

echo "$demo" | c1
```

## Output

```
a1
b1
c1
```
