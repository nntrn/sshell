## Description

Options are:
-v: Selected lines are those not matching any of the specified patterns
-F: Fixed string search
-x: Exact match
-f: Use a file for patterns

## Code

```bash
grep -vFxf file1.txt file2.txt
```

## Example

```
$ cat <<EOF >first.txt
a
b
c
EOF

$ cat <<EOF >second.txt
a
b
1
4
EOF

$ grep -vFxf first.txt second.txt
1
4
```
