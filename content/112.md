## Code

```bash
bash -c 'echo $$ ; ls -l /proc/self ; echo foo'
```

## Example

```
$ bash -c 'echo $$ ; ls -l /proc/self ; echo foo ; echo $$'
68800
lrwxrwxrwx 1 root root 0 Jun 28 05:35 /proc/self -> 68801
foo
68800
```
