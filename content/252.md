## Description

prereq:
npm install -g  @prettier/plugin-xml prettier

## Code

```bash
alias prettier-xml='prettier --bracket-same-line true --xml-whitespace-sensitivity ignore --tab-width 2 --print-width 120 --stdin-filepath  *.xml'
```

## Example

```
cat file.xml | prettier-xml
```
