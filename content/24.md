## Code

```bash
sed "s@+@ @g;s@%@\\\\x@g" | xargs -0 printf "%b"
```

## Example

```
alias urldecode='sed "s@+@ @g;s@%@\\\\x@g" | xargs -0 printf "%b"'

echo "http%3A%2F%2Fwww" | urldecode
http://www
```

## Output

```
http://www
```
