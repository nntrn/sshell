## Code

```bash
csplit --suppress-matched $FILENAME '/^$/' '{*}'
```

## Example

```
$ cat <<EOF >file.txt
  one

  two

  three
EOF

$ csplit --suppress-matched --prefix 'file-' file.txt '/^$/' '{*}'

$ ls file-*
file-00  file-01  file-02

$ find . -type f -name "file-*" -print -exec cat {} \;
./file-00
  one
./file-01
  two
./file-02
  three
```
