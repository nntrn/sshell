## Code

```bash
stat --printf='%U\n%G\n%z\n' $FILEORDIR
```

## Example

```
stat --printf='%U\n%G\n%z\n' .git
```

## Output

```
annie_tran
wheel
2022-03-08 18:39:51.177925946 -0600
```
