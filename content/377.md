## Code

```bash
name="Annie Tran"
echo ${name: -3}
```

## Example

```
$ name="Annie Tran"

$ echo ${name: -3}
ran

$ echo ${name: 3}
ie Tran

$ echo ${name:0:3}
Ann
```
