## Code

```bash
<list of files> | zip name.zip --names-stdin
```

## Example

```
# zip files smaller than 2M
find /var/log -type f -not -size +2M | zip name.zip --names-stdin

# zip only shell scripts
ls -1 *.sh | zip shell-files.zip --names-stdin
```
